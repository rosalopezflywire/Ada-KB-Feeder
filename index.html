GET Fetch existing articles from Ada

Run this in Postman to get all existing articles in your knowledge source, then paste the response below so the tool can detect duplicates.

1

Open Postman and create a new GET request with this URL:

https://flywire-generative.ada.support/api/v2/knowledge/articles/?knowledge_source_id=6662dbfb9663f8c4d106c198
2

In Authorization, select type Bearer Token and paste your Ada token.

3

Click Send. You should get a 200 OK response with a JSON list of articles.

4

Copy the full JSON response and paste it below, then click Import IDs.

POST Upload articles to Ada via Postman

Once you've converted your articles and downloaded the JSON, use Postman to upload them to Ada.

1

Go to the Upload tab, upload your ZIP and click Convert & preview. Review and select the articles you want to upload, then click Download JSON for Postman.

2

In Postman, create a new POST request with this URL:

https://flywire-generative.ada.support/api/v2/knowledge/bulk/articles/
3

In Authorization, select type Bearer Token and paste your Ada token.

4

Go to Body → select raw → set format to JSON. Open the downloaded ada-articles.json file, copy its full content, and paste it into the body.

5

Click Send. A 200 OK response means your articles were created or updated successfully. Each item in the response will show "success": true and whether it was "created": true (new) or false (updated).
